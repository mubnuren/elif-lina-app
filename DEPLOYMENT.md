# Vercel Deployment

Bu paket mevcut GitHub reposuna yüklendiğinde Vercel otomatik deploy alır.

## Güncelleme adımları

1. ZIP dosyasını açın.
2. `elif-lina-app-v13-ultra` klasörünün içine girin.
3. İçindeki tüm dosya ve klasörleri seçin.
4. GitHub'daki mevcut `elif-lina-app` reposunda Add file > Upload files yapın.
5. Dosyaları sürükleyin.
6. Commit changes deyin.
7. Vercel Deployments ekranında yeni build'in Ready olmasını bekleyin.
8. Canlı linki Command + Shift + R ile yenileyin.

Canlı link:
https://elif-lina-app.vercel.app
